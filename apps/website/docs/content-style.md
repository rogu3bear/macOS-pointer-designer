# Content Style Guide

## Brand Assets
- **Accent Color**: `#3B82F6` (Blue).
- **Background**: White (`#FFFFFF`) or Alt (`#F3F4F6`).
- **Text**: Dark Gray (`#171717`) or Muted (`#737373`).
- **Font**: System UI stack (`system-ui`, `-apple-system`, etc.).

## Tone
- Professional, concise, and Mac-idiomatic.
- Avoid marketing fluff. Focus on utility and clarity.
- "WindowDrop" is always CamelCase.

## Layout
- Max-width: 960px.
- Centered content.
- Generous whitespace (`2rem`, `4rem`).
